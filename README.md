# 📊 Product Sales Analysis Dashboard – Power BI

## 📷 Dashboard Preview

![Product Sales Dashboard](Images/Screenshot%202026-06-15%20125824.png)

## An interactive Power BI dashboard designed to analyze product sales performance, revenue trends, customer insights, regional contribution, and profitability metrics.

## 🚀 Project Overview

This dashboard provides a complete overview of sales performance across different regions, categories, customers, products, and time periods.
It helps businesses track KPIs, identify profitable products, monitor customer trends, and make data-driven decisions.

## 📌 Key Features
📈 Monthly Revenue Trend Analysis

🛍️ Revenue by Product Category

🌍 Top Performing States Visualization

💰 Profitability Analysis by Month

👥 Top Customers Revenue Tracking

🏆 Top Selling Products Analysis

📅 Revenue by Day of Week

🌎 Regional Revenue Contribution

📊 KPI Cards for Revenue, Orders, Profit & AOV

🎛️ Interactive Filters (Date, Region, State, City, Category)
